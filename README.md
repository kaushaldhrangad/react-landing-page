## Technology used
- React.js
- Tailwind CSS for the styling
- Framer motion for the Animation

## How to run the app

### Step - 1:- Go to project folder
```
cd project-folder
```
### Step - 2:- install all the dependencies
```
npm install
```

### Step - 3:- Run the Project
```
npm run dev
```
OR
```
yarn run dev
```
